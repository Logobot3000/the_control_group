# The Control Group

The control group is a 2D, online, multiplayer party game made for creating and bonding friends through fast-paced and fun minigames. Created as an 11th grade entrepreneurship project, this game was made for bringing school students together!

The game should release on **March 24, 2026** on Steam for $4.99, and it will be playable at Northview High School's Market Expo on **March 27, 2026** for $1/round.
